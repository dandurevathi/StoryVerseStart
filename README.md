# StoryLearn

**A story-based learning platform for 10th grade students — where concepts are taught through narrative, not textbook prose.**

## Overview

StoryLearn is an educational web application designed for Class 10 students, currently focusing on History and Biology. Instead of presenting concepts as definitions and bullet points, every topic is taught through a short, human-written story — conversational, exam-aligned, and built for the way students actually read.

Each story includes:
- A clear learning objective
- Concept explanation in simple, narrative language
- Pronunciation guides for difficult terms
- A quiz to test understanding
- A badge that students earn on completion

The platform is live with 20+ stories across History, Biology, Chemistry, Civics and Literature, with a teacher-facing dashboard for managing and publishing content.

---

## The Problem This Addresses

Students have two options today — textbooks or YouTube. Textbooks are thorough but dry. Videos are engaging but passive. Neither guarantees that a concept actually sticks.

StoryLearn sits in between: content that is fast to read, human in tone, and structured so that a student finishing a story walks away understanding both *what* happened and *why* it matters — whether they're reading it for the first time or the night before an exam.

## Current Status

This is a working, live version — not a prototype. It is being actively used and iterated on.

The core content format (story + quiz) is functional and tested. The bigger open question being explored right now is whether simplified text alone is sufficient for deep learning, or whether the format needs to become more visual, interactive and student-driven.

This repository documents that ongoing thinking.

## Project Philosophy

> *"Even textbooks can be clearer than a bad explanation. The goal isn't simpler text — it's a format where the student's brain is doing something, not just reading."*

This project is part of a broader personal exploration into what makes concepts genuinely stick — across storytelling, visual learning, interactivity and spaced recall. StoryLearn is one experiment in that direction. Not the last.

## What Is Being Explored Next
The next phase moves beyond text-based stories toward **interactive visual learning** — formats where students don't just read about a concept but actively explore it.

Ideas currently in development:

| Format | Description |
|---|---|
| Explorable scenes | Click on elements of a historical image to uncover the story behind each one |
| Decision moments | At key points in a story, students make a choice — then see what actually happened |
| Live concept maps | Visual diagrams that build as the student reads through the story |
| Misconception challenges | Students commit to an answer before the concept is explained — being wrong first makes the correction memorable |
| Teach-it-back mode | After reading, an AI plays a confused student — the reader has to explain the concept back |

The goal is a format that works for first-time learning, classroom use and last-minute revision — without relying on expensive video production or subscriptions.
